# めぐり — お金を美しく巡らせる

お金を「いくら使ったか」だけでなく、「**どんな気持ちで巡らせたか**」を見つめる、
小さくて静かなアプリです。暮らし・美・家計簿を、やさしく一つにしています。

---

## このアプリの大事な性質（なぜ「ずっと使える」のか）

- **たった数個のファイル**だけで動きます。中心は `index.html` の **1枚**。
- 外のサーバーやデータベース、他社サービスに**いっさい頼っていません**。
  だから「あのサービスが終わって動かなくなった」が起きません。
- **インターネットが無くても動きます**（一度開けば、スマホの中で動き続けます）。
- あなたの**お金の記録は、あなたの端末の中だけ**に保存されます。
  作者も含め、他の誰も見ることはできません。

> つまり、**このフォルダのファイルさえ持っていれば、アプリは永遠に生き続けます。**

---

## ファイルの中身（この一式をいつも一緒に置いてください）

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（これだけでも動きます） |
| `manifest.webmanifest` | ホーム画面に入れるときの設定 |
| `sw.js` | オフラインでも動くための仕組み |
| `icon-192.png` / `icon-512.png` / `apple-touch-icon.png` | アイコン |
| `README.md` | この説明書 |

---

## 使い方（スマホ）

1. 公開した住所（URL）を、スマホのブラウザで開く。
2. **ホーム画面に追加**する。
   - iPhone（Safari）: 共有ボタン → 「ホーム画面に追加」
   - Android（Chrome）: 右上「︙」→ 「ホーム画面に追加」
3. あとは、ホームのアイコンから本物のアプリのように使えます。

---

## 記録のバックアップ・引っ越し

記録は端末の中にあります。**機種変更や、住所（URL）を変えるとき**は、
アプリ内の機能で持ち運べます。

1. 古いほうのアプリ →「巡り」タブ →「**保存する**」（ファイルが1つ書き出される）
2. 新しいほうのアプリ →「**戻す**」→ そのファイルを読み込む

---

## 置き場所を変えたいとき（例: エックスサーバーへ引っ越し）

このフォルダの**ファイル一式**を、サーバーの公開フォルダにアップロードするだけです。
（例: `public_html/meguri/` を作って、その中へ全部入れる）

そのあと `https://あなたのドメイン/meguri/` で開けます。
※ 必ず `https://` で。アイコンや相対パスは引っ越し先でもそのまま動くように作ってあります。

## 更新したいとき

`index.html` を直して、同じ場所に上書きアップロードするだけ。
ネットがあるときに開けば、新しい内容が自然に反映されます（あなたの記録は消えません）。
